# Ranking de Fidelidade

👉 ****⭐Funcionalidade**
  [Fidelidade Unificada](https://www.notion.so/24300ee50e9081e08a82c76beedebbfa)


👉 ****⚙Config**
O ranking de fidelidade está disponível em **Delivery > Ranking de fidelidade **no painel de relatórios.

## 😎Como funciona

O ranking de fidelidade mostrará todos os clientes que estão participando da fidelidade ativa. Por ele é possível visualizar o andamento de cada cliente mostrando a quantidade de pedidos e quantos faltam. Também é possível clicar em **relatório resumido **para saber apenas a quantidade de clientes em cada etapa ao invés de uma linha para cada cliente.
