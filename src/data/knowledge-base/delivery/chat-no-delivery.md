  # Chat no delivery

  ## Neste artigo, você aprenderá a ativar a funcionalidade de chat em seu sistema de delivery, permitindo que os clientes se comuniquem diretamente com a loja através do link de pedidos.

### 💬 Como funciona o Chat do Delivery?

Com o chat ativado, seu estabelecimento pode se comunicar diretamente com o cliente após o pedido ser feito — tudo dentro do próprio sistema.

Essa funcionalidade é ideal para:

Avisar que o pedido está a caminho;

Confirmar alguma informação com o cliente;

Facilitar a comunicação sem precisar usar outro canal.

### ✅ Como ativar o chat

Vá em **Ajustes > Configurações**;

Ative a opção **Habilitar chat**.

[Imagem]

### 📱 Como aparece no sistema (para o estabelecimento)

No painel de pedidos do delivery, ao clicar sobre um pedido, você verá a opção **Chat com o cliente**. Basta clicar para iniciar a conversa.

Essa opção aparece dentro de cada pedido, de forma prática.

[Imagem]

### 👤 Como aparece para o cliente

Assim que o pedido for feito, o cliente verá a opção **Chat com a loja** no link de acompanhamento do pedido.

Ele poderá usar esse botão para conversar diretamente com você, caso queira tirar dúvidas ou acompanhar a entrega.
