# Copiar produto existente | Delivery
  # Copiar produto existente | Delivery

  ## Descubra, neste artigo, o passo a passo para duplicar produtos já cadastrados no seu sistema de delivery.

  ### Como funciona

Essa opção facilita a criação do cardápio do delivery copiando produtos que já existem no seu PDV (caixa).

⚠️ Essa opção **só aparece ao cadastrar um novo produto** no delivery.

### ✅ Como usar a função "Copiar produto existente"

No cardápio do Delivery, clique em **+ novo produto** e, em seguida, no botão **Copiar um produto existente**.

[Imagem]

- O sistema vai abrir a lista de produtos do seu **PDV**.Use a barra de busca para encontrar o produto que deseja copiar.

- ✅ Clique no produto desejado para selecioná-lo.
[Imagem]

- Pronto! O sistema vai preencher automaticamente as seguintes informações no produto do delivery:

Foto do produto

Nome

Preço

Complementos (se houver)

Essa função é ideal para quem quer economizar tempo na hora de cadastrar os produtos no delivery.
