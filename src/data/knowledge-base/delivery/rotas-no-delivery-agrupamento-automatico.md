### 🛵 O que são Rotas Automáticas no Delivery?

A funcionalidade de rotas automáticas permite **agrupar pedidos por proximidade e tempo de entrega**, formando rotas de entrega inteligentes. Isso facilita o trabalho dos entregadores e otimiza o tempo das entregas.

[Imagem]

⚠️ **Importante**: essa função só está disponível para quem usa o modo de entrega por [**raio**](https://21701426.hs-sites.com/ajuda-yooga/%C3%A1rea-de-entrega-por-raio?hsLang=pt-br).

### ⚙️ Como ativar e configurar as Rotas Automáticas?

#### 1. Ative a função no sistema

- Acesse **Ajustes > configurações;
- Ative a função **Habilitar agrupamentos automáticos.
### [Imagem]

#### 2. Configure as regras de agrupamento

Acesse **Ajustes > Configurações**;

Vá até a opção **Configurações do Delivery**;

Na aba **Agrupamento Automático**, preencha os seguintes campos:

[Imagem]

|
Campo |
O que significa |
 |

|
**Número máximo de pedidos** |
Quantos pedidos podem ser agrupados em uma mesma rota. |
 |

|
**Distância máxima (KM)** |
Distância máxima, em km, entre os pedidos para entrarem na mesma rota. |
 |

|
**Tempo máximo (Min)** |
Tempo máximo de entrega permitido entre os pedidos agrupados. |
 |

📝 Após salvar, os pedidos que se encaixarem nas regras acima serão agrupados automaticamente e exibidos com uma etiqueta de rota.

### 📦 Como despachar ou finalizar todos os pedidos da rota?

Acesse o **Delivery **> **Mapa de pedidos;
[Imagem]

Encontre a rota desejada;

Clique em **Despachar** para iniciar a entrega de todos os pedidos dessa rota;

[Imagem]

Após a entrega, clique em **Finalizar** para encerrar a rota.

[Imagem]
