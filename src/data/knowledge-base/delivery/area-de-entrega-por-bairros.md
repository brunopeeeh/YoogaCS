### Ativando a área de entrega por Bairros

Se a sua área de entrega estiver configurada para Raios, é necessário desativar esse modo. Para isso, siga os passos abaixo:

Clique em **Ajustes** > **Configurações** dentro do tópico Delivery.

[Imagem]

Em seguida, desative a opção "Habilitar área de entrega raio" e, por fim, clique em "Salvar".

[Imagem]

Prontinho, agora vamos para o próximo passo! 😇

### Como cadastrar os Bairros
Para iniciar o cadastro de Bairros em sua área de entrega, clique em Ajustes > Bairros.

[Imagem]

Agora, preencha os dados conforme solicitado para garantir que todas as informações sejam registradas corretamente.

**Campos
**Bairro: **Insira o nome do bairro exatamente como aparece no Google Maps, para garantir que a localização esteja correta.

**Taxa de entrega:** Informe o valor que será cobrado para a entrega nesse bairro, assegurando que os clientes tenham clareza sobre os custos.

**Tempo de entrega:** Especifique o tempo estimado para realizar a entrega nesse bairro, ajudando os clientes a se programarem adequadamente.

[Imagem]

Não se esqueça de clicar em **+ Adicionar bairro** para salvar o bairro! 😉
