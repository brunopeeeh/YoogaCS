### O que é a API de Conversões?

A API de Conversões é uma ferramenta da Meta que permite rastrear o comportamento dos usuários em sites. Com ela, é possível mensurar como os clientes acessam e interagem com o link do seu cardápio em um domínio próprio, garantindo maior confiabilidade nos dados enviados ao Facebook.

###

### Quais dados são necessários para o rastreio?

Para utilizar a API de Conversões, são necessários apenas dois dados:

**Pixel ID
**Access Token
Essas informações autorizam o envio seguro dos eventos diretamente pelo servidor.

###

### Onde posso obter o Pixel ID e o Access Token?

**Pixel ID:** disponível no **Gerenciador de Eventos da Meta**.

**Access Token:** gerado na seção de **API de Conversões**, dentro das configurações do pixel.

###

### Qual a diferença entre o Pixel e a API de Conversões?

**Pixel:** funciona no navegador do usuário.

**API de Conversões:** funciona no servidor.
Quando utilizados em conjunto, proporcionam maior precisão, consistência e confiabilidade nos dados coletados.

###

### Por que utilizar a API de Conversões?

Apoia decisões mais estratégicas para o seu negócio

Permite monitorar o desempenho dos clientes no delivery

Garante mais controle e segurança sobre os seus dados
