### O que é o Google Tag Manager?

O Google Tag Manager (GTM) é uma ferramenta do Google que facilita a gestão de códigos de rastreamento no seu site ou cardápio de delivery. Em vez de inserir vários códigos diretamente no site, o GTM centraliza tudo em um único painel, garantindo mais organização e praticidade. Ele não armazena dados nem gera relatórios — sua função é apenas enviar informações para outras plataformas.

### Qual código é necessário para o rastreio?

Para que o GTM funcione, é necessário o **ID do container**, identificado no formato **GTM-XXXX**. Esse código define quais eventos e integrações serão executados no site ou no delivery. Sem ele, nenhuma configuração de rastreamento será ativada.

###

### Onde encontro o ID do Google Tag Manager?

O ID do GTM pode ser encontrado dentro da sua conta do Google Tag Manager, logo após a criação do container. Ele aparece no topo da tela e também nas instruções de instalação fornecidas pela própria ferramenta.

### Por que o Google Tag Manager é importante?

O GTM traz mais agilidade e organização para o seu negócio. Ele permite adicionar ou ajustar eventos de rastreamento sem a necessidade de alterar o código do site, reduzindo erros técnicos e a dependência de desenvolvedores. Além disso, garante que os dados enviados para ferramentas como Google Analytics, Facebook Pixel e API de Conversões sigam um padrão único.

###

### Qual a diferença entre Google Tag Manager e Google Analytics?

O Google Tag Manager apenas dispara eventos e integrações. Já o Google Analytics é responsável por receber esses eventos, armazenar os dados e permitir a análise do comportamento dos clientes.

### Por que utilizar o Google Tag Manager?

Apoia decisões mais estratégicas para o seu negócio

Permite o monitoramento de eventos importantes no seu delivery

Garante mais controle e padronização dos seus dados
