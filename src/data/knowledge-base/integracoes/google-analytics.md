### O que é o Google Analytics?
O Google Analytics é uma ferramenta do Google que analisa o acesso e o comportamento dos visitantes. No delivery, ele ajuda a entender como os clientes chegam ao cardápio, quais páginas visitam, quanto tempo permanecem e quais ações realizam antes de tentar fazer um pedido.

###

### Para que serve o Google Analytics no delivery?
Ele permite acompanhar o desempenho do seu cardápio online, identificar o interesse dos clientes e entender quais canais e campanhas trazem mais resultados.

###

### Qual código é necessário para o rastreamento?
O Google Analytics 4 (GA4) utiliza um ID de mensuração no formato **G-XXXX**. Esse código garante que os dados do site ou página de delivery sejam enviados corretamente para sua conta do Google Analytics.

###

### Onde posso encontrar o ID G-XXXX?
Você pode localizar esse código no Google Analytics, acessando:
**Administrador → Propriedade → Fluxos de dados → Web**.
Esse é o ID que deve ser informado ou integrado ao seu cardápio.

###

### Por que o Google Analytics é importante para o meu negócio?
Com ele, você consegue responder perguntas importantes, como:

De onde vêm seus clientes

Quais campanhas têm melhor desempenho

Quais horários geram mais acessos

Quais canais despertam mais interesse

Essas informações ajudam a direcionar melhor seus investimentos e estratégias de marketing.

###

### Qual a diferença entre Google Analytics e Google Tag Manager (GTM)?
O Google Analytics é onde os dados são analisados e transformados em relatórios, já o Google Tag Manager é apenas a ferramenta técnica usada para enviar esses dados e eventos para o Analytics.

###

### Por que devo usar o Google Analytics?
Para acompanhar suas campanhas em tempo real

Para entender o comportamento dos clientes no delivery

Para tomar decisões mais estratégicas com base em dados reais
