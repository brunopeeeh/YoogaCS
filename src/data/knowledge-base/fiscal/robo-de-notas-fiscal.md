  # Robô de notas | Fiscal

  ## Veja, nesse artigo, como funciona e como acessar seu robô de notas.

  ### Como funciona

O Robô de Notas da Yooga é responsável por incluir de forma automática os XML's das notas de compra emitidas no seu CNPJ, ou seja, toda vez que você realiza uma compra e informa o CNPJ, o sistema deixa os dados dela disponível para que você faça a entrada no estoque dos seus produtos e gere despesas no seu financeiro.

### Como acessar

Para visualizar as notas adquiridas pelo Robô de Notas, basta acessar seu Dashboard no link [_**painel.yooga.com.br**_](http://painel.yooga.com.br/)

Dentro do painel acesse **Fiscal > Notas de compra;
[Imagem]

Nessa Aba você terá a opção de Baixar o XML's das Notas ou fazer uma entrada de nota, gerando novos produtos ou adicionando estoque aos produtos já existentes no seu sistema.
