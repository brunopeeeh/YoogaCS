  # Como baixar XML's das notas emitidas | Fiscal

  ## Aprenda, neste artigo, o passo a passo para baixar os arquivos XML das notas fiscais emitidas e como enviá-los corretamente para a sua contabilidade.

  ### Tópicos

[#enviarcontabilidade](Como enviar para contabilidade)

[#enviarcontabilidade](Como baixar no dispositivo)

### Como enviar para contabilidade

Acesse a área fiscal do seu painel de relatórios [Clicando aqui](http://painel.yooga.com.br/)!

Após realizar o acesso clique em **Fiscal > vendas;
**[Imagem]
Filtre pelo Período desejado, modelo e status;

[Imagem]

Agora clique em Enviar para contador, digite o e-mail que deve ser feito o envio e clique no ícone do avião de papel para finalizar o envio.

[Imagem][Imagem]

### Como baixar no dispositivo

Acesse a área fiscal do seu painel de relatórios [Clicando aqui](http://painel.yooga.com.br/)!

Após realizar o acesso clique em **Fiscal > vendas;
**[Imagem]
Filtre pelo Período desejado, modelo e status;

[Imagem]

Agora clique em Download de todas. Caso possua uma grande quantidade de notas transmitidas para realizar download, seu arquivo de notas será disponibilizado na aba **Solicitações de XML's, **na tela apresentará a mensagem abaixo, clique em **Verificar Status[Imagem]
Com o status **Sucesso** é possível realizar o download do arquivo clicando no ícone ao lado direito ⬇️

[Imagem]
