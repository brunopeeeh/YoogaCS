# Eu preciso emitir notas fiscais?
  # Eu preciso emitir notas fiscais?

  ## Como saber se precisa emitir notas fiscais.

  É necessário fazer a emissão de notas fiscais empresas optantes pelo [Simples Nacional](http://www8.receita.fazenda.gov.br/SimplesNacional/) ou que utilizam o _**Regime Normal **q_ue efetuam operações de venda para pessoas físicas e jurídicas ou atividade de transformação para fábricas e industrias.

Atualmente na Yooga é possível emitir NFCe, NF-e, mas somente para clientes optantes pelo Simples Nacional.

Se você não tem certeza se está enquadrado nesses regimes e não sabe se deve ou não emitir notas fiscais, procure pela Secretaria da Fazenda. Entretanto, é recomendado optar sempre pela emissão, já que a maioria das empresas já são obrigadas a emitir.

Se ainda tiver dúvidas não deixe de nos acionar através do [//ajuda.yooga.com.br?hsLang=pt-br](suporte)!
