# Cliente

A aba de Clientes está disponível no [painel.yooga.com.br](http://painel.yooga.com.br/) **> Cadastros > Clientes**.

## 😎Como funciona

Nessa aba é possível cadastrar os clientes em **Novo Cliente**, editar as informações e excluir os clientes já cadastrados.
