## 😉O que é

O **DRE (demonstrativo de resultado de exercício)** é um relatório contábil que evidencia se as operações de uma empresa estão gerando lucro ou prejuízo, considerando um determinado período.

Ela é composta por várias linhas que indicam os valores de vendas, custos, impostos, lucro bruto, lucro operacional e lucro líquido.

## De onde são puxadas as informações

As informações são puxadas do relatório de **lançamentos **do financeiro. Ao clicar em configurar seguindo o tópico anterior, as categorias que aparecerão para configuração serão as categorias de lançamentos do financeiro.

## Como configurar o CMV no DRE

O sistema possui uma categoria gerada automaticamente para o CMV, através das vendas realizadas no sistema. Para isso é preciso que todos os produtos, complementos e insumos possuam o preço de custo cadastrados corretamente.



O custo é enviado para essa categoria após o fechamento de cada caixa, seguindo o fluxo:

- As vendas são feitas no sistema;
- o caixa é fechado;
- O custo dos produtos vendidos nesse caixa é enviado para a categoria **sistema - custo de mercadoria **no DRE.
