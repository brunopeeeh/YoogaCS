# Venda por vendedor

O relatório de venda por vendedor está disponível em **relatórios > venda por vendedor **no painel de relatórios. Precisa estar com a config abaixo ativa para aparecer.

👉 ****Config**
## 😎Como funciona

Ele é utilizado para casos em que o dono do estabelecimento realiza o pagamento de comissão conforme as vendas realizadas por cada vendedor, nesse caso após ativado terá disponível esse relatório informando a quantidade de vendas, valor faturado e comissão a ser paga no período filtrado.
