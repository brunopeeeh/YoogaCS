## Dúvidas Frequentes

👉 **⁉ **O que significa e no que interfere cada data inserida na janela de receita/despesa?**
  - **Competência: **Essa data é referente ao mês a qual a receita/despesa é referente ou foi gerada.
  - **Vencimento: **É a data limite para o pagamento/recebimento da despesa/receita.  É nessa data específica que o lançamento vai aparecer antes de ser clicada a opção Receber/pagar.
  - **Pago/recebido: **É a data que o valor da despesa/receita foi paga/recebida. Essa data vai definir onde os valores vão aparecer nos lançamentos.


  **Exemplo 1:** Nesse exemplo, o lançamento vai aparecer no mês 07/23, pois a data de **Vencimento **foi colocada nesse dia e ainda não há data de Recebimento (canto esquerdo inferior).
