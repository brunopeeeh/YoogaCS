# Categorias

> [!NOTE]
> As categorias citadas nessa página são referentes ao cardápio PDV do sistema

A aba de Categorias está disponível no [painel.yooga.com.br](http://painel.yooga.com.br/) **> Cadastros > Categoria**.

## 😎Como funciona

Nessa aba é possível cadastrar as categorias para o cardápio do PDV. É dentro das categorias que os produtos estão inseridos.
Ex.: Categoria: Bebidas, Pizzas, Hambúrgueres.

Clicando em **Nova Categoria **é possível criar as categorias.
