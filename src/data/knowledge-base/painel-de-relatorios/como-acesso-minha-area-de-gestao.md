# Como acesso minha área de gestão?
  # Como acesso minha área de gestão?

  ## Por aqui você fica sabendo todos os passos para que você acesse o seu Painel Gestor!


O Painel Gestor está disponível apenas para os planos Essencial, Completo e Premium, sendo inacessível no plano Básico. Além disso, somente usuários com permissão de administrador podem acessar essa área.

O Painel Gestor é a área por onde você consegue verificar as informações do seu App Yooga (o sistema de pedidos), além de gerir toda a sua parte financeira, estoque, entre outras informações.

O acesso ao Painel se dá pelo link [painel.yooga.com.br](http://painel.yooga.com.br)no seu navegador, que pode ser o Google Chrome, Microsoft Edge, Safari, Mozila, entre outros

[Imagem]

**Ponto de atenção:** o login e a senha de acessos ao Painel Gestor são as mesmas utilizadas no App Yooga (o sistema de pedidos)
