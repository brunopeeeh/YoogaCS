# Produtos

> [!NOTE]
> Os produtos citadas nessa página são referentes ao cardápio PDV do sistema


A aba de Produtos está disponível no [painel.yooga.com.br](http://painel.yooga.com.br/) **> Cadastros > Produtos**.

## 😎Como funciona

Nessa aba é possível cadastrar os produtos para o cardápio do PDV.

🔹CÓD: É o código PDV do produto.

🔹 NOME PRODUTO: É o nome dado ao produto

🔹 VALOR DE CUSTO: É o valor que foi gasto pelo estabelecimento para fazer/comprar o produto.

🔹 VALOR DE VENDA: É o valor de venda do produto no PDV.

🔹 ESTOQUE: É a quantidade de itens que estão em estoque no momento da visualização.

FAQ

👉 **⁉** Excluí um produto, como vão ficar os Relatórios de Produtos?**
  **Resposta: **Quando uma categoria é excluída, os produtos que foram vendidos antes de serem excluídos continuam aparecendo no Relatório de Vendas por Produtos/Categoria e no Relatório de Vendas por Produtos.


👉 **⁉** Para que serve o Código de Barras (EAN)?**
  **Resposta: **É possível inserir o Código de Barras (EAN) no produto, dessa forma a venda pode ser realizada utilizando-se um Leitor de códigos de barras.
