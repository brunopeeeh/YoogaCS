# Adicionar App Yooga em iPhone
  # Adicionar App Yooga em iPhone

  ## Neste artigo, você aprenderá o passo a passo para adicionar o aplicativo da Yooga em seu dispositivo iOS.

  Atualmente, não dispomos de um aplicativo específico para iOS, mas você pode criar um atalho direto na tela inicial do seu iPhone. Para isso, siga estas etapas:

1. Abra o navegador Safari e acesse o site app.yooga.com.br.

2. Toque no ícone de compartilhamento, que se parece com um quadrado contendo uma seta apontando para cima.

[Imagem]

3. Selecione a opção "Adicionar à Tela de Início".

[Imagem]

4. Confirme o nome que deseja para o atalho e toque em Adicionar.

[Imagem]

Pronto! Agora, o ícone aparecerá na sua tela inicial, como se fosse um aplicativo comum 💙.
