# Devolução de Vendas ( Cliente devolvendo o produto para loja )

**Casos em que a devolução se enquadra:
- Cliente comprou um produto e preciso devolver porque comprou errado.
- A mercadoria veio com defeito, quebrada, avariada.

**APP/DESK/Emissor
**Natureza da Operação:
Devolução de Vendas - Devolução Completa

Devolução parcial de Vendas- Somente alguns itens da nota

**Finalidade:** Devolução de Mercadoria.

**Tipo de Operação:** Entrada.

**CFOP:** Mais usados pelos clientes


**1202 **- **Devolução de venda
**2202 **- **Devolução de venda para outro estado
**1403 **- **Devolução de venda de mercadoria ST (substituição tributária)
**2403 **- **Devolução de venda de mercadoria ST para outro estado
**CSOSN**:

**900** - Significa outras saídas, essa CSOSN destaca o ICMS informado nos produtos.
