# Cadastro de complementos | Cardápio QR code

  ## Neste artigo, você aprenderá o passo a passo para cadastrar complementos de produtos no cardápio QR code

  Primeiro clique em Ajustes > Cardápio QR Code em seguida selecione o produto que terá os complementos adicionados;
[Imagem]

**[Imagem]
Caso a categoria de complemento já exista e deseja apenas adicionar mais complementos a ela, pule o passo de criação de categoria de complementos.

Dentro do produto, acessa área de complementos e clique em **+ Nova Categoria

**[Imagem]

Insira o nome na categoria de Complemento criada, em seguida clique em **+ Adicionar Complemento.
Caso a seleção dos complementos sejam obrigatórias devem selecionar  **Preenchimento Obrigatório** e inserir **Quantidade Mínima** e **Máxima** de seleção.

[Imagem]

Insira os dados de Nome do Complemento, Preço e Descrição conforme deseja.

[Imagem]

Prontinho, agora só Salvar! 😇

Caso queira cadastrar mais categorias de complementos, deve repetir o processo para cadastrar separadamente as opções de categorias e de complementos conforme desejar.
