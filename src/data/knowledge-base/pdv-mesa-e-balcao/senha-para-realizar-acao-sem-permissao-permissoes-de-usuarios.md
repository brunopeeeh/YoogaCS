# Senha para realizar ação sem permissão | Permissões de Usuários

  ## Veja nesse artigo como configurar uma senha para toda vez que um usuário tentar realizar uma ação sem permissão.


### Tópicos

[#comofunciona](Como funciona a senha para ações sem permissão)
[#comoconfigurar](Como configurar)
[#usonapratica](Uso na prática)

Essa função é limitada ao módulo de vendas Mesas e Balcão, no módulo de vendas Somente Balcão não se aplica.

### Como funciona a **Senha para ações sem permissão
Ao ativar essa função, sempre que um usuário tentar realizar alguma ação que não está permitida para ele, o sistema vai solicitar uma senha de 4 dígitos na tela. Essa senha deve ser cadastrada em um usuário que tenha permissão para a ação — normalmente o gerente ou administrador do sistema.

Assim, você mantém o controle do que pode ou não ser feito por cada perfil da equipe, sem bloquear totalmente a operação.

### Como configurar

A configuração é feita em 3 etapas simples:

[#ative](**Ativar a função;**)

[#ajuste](**Ajustar as permissões dos usuários que deverão solicitar senha;**)

[#cadastre](**Definir a senha no usuário autorizado (gerente ou administrador).**)

#### 1. Ative a função

Acesse o menu **Ajustes > Geral** e ative a opção **Senha para realizar ação sem permissão**.

[Imagem]

#### 2. Ajuste as permissões dos usuários

Acesse o menu **Usuários**, clique no usuário desejado e vá até a aba **Permissões**. Desative as ações que devem exigir a senha.

[Imagem]

[Imagem]Repita esse processo para cada usuário que **não deve ter permissão direta** para essas ações.

#### 3. Cadastre a senha no usuário autorizado

Agora, volte à lista de usuários e clique naquele que **terá a senha** (geralmente o gerente).Dentro do cadastro dele, insira uma senha de 4 dígitos no campo **Senha de permissão**.

[Imagem]

#### Uso na prática

Imagine a seguinte situação:

Um garçom tenta **excluir uma mesa**, mas essa ação está desativada nas permissões dele;

O sistema vai pedir uma senha;

O garçom chama o gerente, que tem a senha cadastrada;

O gerente insere a senha e a ação é liberada na hora.
