## 😎Como configurar

**1.  **Para configurar a impressora no App Yooga, basta instalar nosso aplicativo no computador acessando o link:[ yooga.com.br/download](http://yooga.com.br/download)

![Imagem](https://fc-use1-00-pics-bkt-00.s3.amazonaws.com/d638ab8ad6b1b60f9bded19d55483e045352b47649fae81ff8ee8c58c1a747a1/f_faq-image/u_65a7f26b29c0a52dff06c49c32d4505653030f5f5c51b692e9763d04aabba606/img_8i051qd0lu_eab79f3493a1e157789085721f1c06f24c42c86b22ef1195900e48bcf4254e85.jpeg)

**2.**  Após a instalação, a extensão de impressão do sistema estará pronta e você pode copiar o token único clicando na** engrenagem -> Geral -> vá até a aba de impressão e clique no token:
![Imagem](https://fc-use1-00-pics-bkt-00.s3.amazonaws.com/d638ab8ad6b1b60f9bded19d55483e045352b47649fae81ff8ee8c58c1a747a1/f_faq-image/u_65a7f26b29c0a52dff06c49c32d4505653030f5f5c51b692e9763d04aabba606/img_7tgkivauqq_3465771367462fb539ee1bbfeb8756c3d2f3d614862587aad4dab88ff511342d.gif)

**3.** Após copiar o token de impressão, clique com o botão direito sobre a nossa extensão que fica no canto inferior direito do computador e clique em configurar socket de impressão:

![Imagem](https://fc-use1-00-pics-bkt-00.s3.amazonaws.com/d638ab8ad6b1b60f9bded19d55483e045352b47649fae81ff8ee8c58c1a747a1/f_faq-image/u_65a7f26b29c0a52dff06c49c32d4505653030f5f5c51b692e9763d04aabba606/img_hslcrn0btp_2898b66f45c454d3430d58decba5b4c76a7ff2b12501f8ea11d549d46a0d0954.png)

Nessa extensão você também configura a impressora padrão do sistema pela opção **impressora padrão**. Note que a extensão ficará com a cor verde, após isso habilite as configurações **Impressão via API **e** Impressão via Socket **dentro do sistema:

![Imagem](https://fc-use1-00-pics-bkt-00.s3.amazonaws.com/d638ab8ad6b1b60f9bded19d55483e045352b47649fae81ff8ee8c58c1a747a1/f_faq-image/u_65a7f26b29c0a52dff06c49c32d4505653030f5f5c51b692e9763d04aabba606/img_oi8ap7asbm_fd4ee07b3e0e9e86d5ed7c2fa57aed1d001279588cc945c07d90433220e0303b.png)

Para configurar a impressora no Delivery, basta ir até a aba Delivery e escolher a impressora padrão, e caso tenha uma segunda impressora a impressora secundária:

![Imagem](https://fc-use1-00-pics-bkt-00.s3.amazonaws.com/d638ab8ad6b1b60f9bded19d55483e045352b47649fae81ff8ee8c58c1a747a1/f_faq-image/u_65a7f26b29c0a52dff06c49c32d4505653030f5f5c51b692e9763d04aabba606/img_5vjga6lnn4_6db6886bf05ed600273728b033efe8f3193a615146fd34741071584abf9e9c42.png)

_Obs: Caso a impressora ainda não tenha sido instalada no computador, é necessário seguir algum dos outros passo a passo disponíveis na aba __**periféricos → impressoras**__ aqui na wiki. 😁💙_
