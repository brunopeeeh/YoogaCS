# Impressora em Generic Text

> [!NOTE]
> Esse modelo de impressão não faz a impressão de notas fiscais modelo NFC-e.
Se o cliente tem fiscal ativado, a configuração deve ser feita pelo driver próprio da impressora.

## Quando usar

Você pode usar a opção gereric text quando a impressora não tem um driver próprio ou não está sendo reconhecido por ele.
