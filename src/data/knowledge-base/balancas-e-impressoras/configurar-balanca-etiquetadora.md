# Configurar balança etiquetadora

Para matar a dúvida de todos vocês definitivamente 😍


**Balanças de leitura de Etiqueta
**Importante saber isso aqui :

- O sistema Yooga não se comunica com a Balança
- A leitura das etiquetas só funciona no modulo **Balcão
- O sistema Yooga não envia carga de produtos para a balança
- Não precisa configurar a balança no sistema Yooga.
- O cliente precisa cadastrar todos os produtos no programa da **balança dele. Não tem vinculo com a Yooga essa parte.
- Cada produto existe um código de PDV no programa da balança.


## Como** configurar o sistema Yooga para ler o código das etiquetas?
Cadastrar os produtos no sistema da balança inserindo os códigos PDV. O código PDV deve ser o mesmo tanto no cadastro dos produtos na Yooga quanto na configuração da balança etiquetadora.


## O**nde pego o código de PDV para testar?
Solicitando o cliente no chat a foto de uma etiqueta para vermos se o código de barras é por **peso **ou **valor**. E perguntar qual é o código de PDV desse produto.
