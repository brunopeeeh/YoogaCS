# Epson | Instalação de impressora no computador
  # Epson | Instalação de impressora no computador

  ## Veja nesse conteúdo como instalar sua impressora Epson no computador

  Caso você ainda não tenha a impressora instalada no seu computador, utilize o driver da Epson para iniciar a instalação, ele pode ser adquirido diretamente no site da fabricante.

Clique no arquivo baixado e siga os passos da instalação:

[Imagem]

Selecione o tipo de comunicação USB ou Ethernet (REDE) e clique em Save Settings para instalar a impressora:

[Imagem]

Caso a impressora esteja conectada via cabo USB, basta clicar em avançar que o instalador vai reconhecer a porta correta e finalizar a instalação.

Caso ela esteja conectada via Rede, é necessário que você obtenha o IP da impressora, desligue o aparelho ⇒ pressione o botão de avanço de papel e ligue ele novamente com o botão pressionado para adquirir o IP da impressora.

Adicione o IP no campo IP Adress e finalize clicando em Save Settings:

[Imagem]

Após finalizar a instalação da impressora em seu computador basta configurá-la no App Yooga seguindo o conteúdo **[//21701426.hs-sites.com/ajuda-yooga/como-configurar-a-impressora-no-app-yooga?hsLang=pt-br](Como configurar a impressora no App Yooga).
